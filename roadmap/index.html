Data Analytics Plan Outline

I am recording data from this site using Google Analytics with Google Tag Manager. Below is my plan for what I would like to record/track. I will use this for my testing and debugging.

  • Track time spent on homepage
  • Track page viewed after homepage
  • Count blog posts viewed out of total
  • Count number of clicks for email address
  • Count number of clicks for GitHub link
  • Track MailChimp sign-ups
